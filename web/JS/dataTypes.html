JavaScript Data Types

Understanding data types is crucial in programming. JavaScript has several built-in data types:

  • String: Represents text. Example: "Hello, World!"
  • Number: Represents both integer and floating-point numbers. Example: 42, 3.14
  • Boolean: Represents logical entities. Can be either true or false.
  • Object: A collection of properties. Example: {name: "Ayush", age: 25}
  • Array: An ordered list of values. Example: [1, 2, 3, 4, 5]
  • Undefined: A variable that has been declared but not assigned a value.
  • Null: Represents the intentional absence of any object value.
  • Symbol: A unique and immutable primitive value used as the key of an object property.
  • BigInt: Represents integers with arbitrary precision. Example: 9007199254740991n

// String

console.log(typeof person); // Output: object
// Array
let numbers = [1, 2, 3, 4, 5];
console.log(typeof numbers); // Output: object
// Undefined
let x;
console.log(typeof x); // Output: undefined
// Null
let y = null;
console.log(typeof y); // Output: object
// Symbol
let sym = Symbol("id");
console.log(typeof sym); // Output: symbol
// BigInt
let bigIntNum = 9007199254740991n;
console.log(typeof bigIntNum); // Output: bigint

Understanding these data types will help you write more effective JavaScript code!
